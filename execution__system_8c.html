Notes:

The module execution system is aware of all compute modules and io devices available to the controller application. Its purpose is to execute compute modules according to configurable execution policies and to link modules with their corresponding platform io devices. The execution system is also an API to each of the modules and io devices, capable of providing platform specific functions for things like system time, execution time, platform details, etc
