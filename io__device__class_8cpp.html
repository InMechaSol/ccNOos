An io device provides a link to the real world from the compute solution. It relies on serialized data of binary or text form. Serialization, physical io device HW, and associated code are left to be platform and application specific. Device compute modules and api compute modules designed to rely on this base io device concept, are then portable to interface with any physical io device HW and utilize any serialization strategy.
